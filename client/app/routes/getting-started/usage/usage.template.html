Usage

Create your application

  • Folder Structure
  • Build
  • Watch

To create your application first create the folder where your aplication will be placed, usualy the name of the folder is the same as the application name. 



Now run the generator.




If you don't want the include the  sample application(this documentation) run the generator with --clean flag.




Follow the on screen prompts and your app will be created.



    
Note

    
Do not use the default module name if you need a specific one because this will be harder to change after the application is created.

Watch


If you want the changes made in the code to appear immediately you can use watch.




This will also run tests every time you change something to disable tests run 





    
Note

    
If you add new routes, components, etc.. sometimes watch doesn not pick them up just stop the process by using
        Ctrl+C and rerun gulp && gulp watch
